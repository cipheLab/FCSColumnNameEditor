# FCSColumnNameEditor
This tool can be used to select the parameters to be anonymised in an FCS file.

   ## Usage

### 1 - Load FCS file(s)
### 2 - Select the lines to be anonymised
### 3 - Click on the ‘change Names’ button
### 4 - Download the modified FCS
